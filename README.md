## 🛠️ QUICK START (DEPLOYMENT)

### 1. Backend Orchestration (Inference)
```powershell
cd api
pip install -r requirements.txt
python main.py
```
*Port: `http://localhost:8000` | Open `/docs` for Swagger UI*

### 2. Frontend Interface (Dashboard)
```powershell
cd webapp
npm install
npm run dev
```
*Port: `http://localhost:5173` | Requires Backend to be running*
