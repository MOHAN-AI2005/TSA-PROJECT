# Multivariate Electricity Demand Forecasting Using Weather and Calendar Effects

## Project Overview
This repository contains a professional-grade research suite for National Electricity Demand Forecasting across India.

![Signals Analysis Graph](./docs/images/signals.png)

### Performance Results
The system utilizes a Gradient Boosting ensemble to achieve high-precision results by incorporating weather and calendar features.

![Model Performance Benchmarks](./docs/images/metrics.png)

### Intelligence & Diagnostics
The system includes built-in stability audits and feature importance analysis to ensure the transparency of the forecast results.

![Intelligence Diagnostics](./docs/images/diagnostics.png)

---

## Repository Structure
- `api/`: FastAPI Backend
- `webapp/`: React Frontend
- `data/`: Processed Grid Data
- `models/`: Trained Model Binaries
- `docs/`: Technical Documentation
- `scripts/`: Implementation Logic

## How to Run
1. Start the API: `cd api; python main.py`
2. Start the App: `cd webapp; npm run dev`
